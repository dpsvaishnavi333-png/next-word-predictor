# Next Word Predictor

This project is a simple Next Word Prediction system built using TensorFlow and Streamlit. It predicts the next word based on the previous input text using a trained language model.

## Features
- Text preprocessing
- Model training using TensorFlow
- Next word prediction
- Interactive web interface using Streamlit

## Project Structure
- data.txt – training dataset
- preprocess.py – text preprocessing
- train.py – model training
- predict.py – word prediction logic
- requirements.txt – required Python libraries

## Installation
1. Clone the repository
2. Create a virtual environment (optional)
3. Install dependencies:
   pip install -r requirements.txt

## Run the Application
streamlit run app.py

## Technologies Used
- Python
- TensorFlow
- Streamlit
